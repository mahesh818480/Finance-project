# 💰 Finance Dashboard (Angular 18)

A modern Finance Dashboard built using Angular 18, Angular Material, and Tailwind CSS.

---

## 🚀 Features

* 🔐 Login & Register (localStorage authentication - (temporary))
* 🛡️ Route Guards for secure navigation
* 📊 Dashboard (Balance, Income, Expense + Charts)
* 💳 Transactions (Add, Delete, Filter, Pagination)
* 📈 Insights Page (analytics & trends)
* 🌙 Dark Mode support
* 📱 Fully Responsive UI (Mobile + Desktop)

---

## 🛠️ Tech Stack

* Angular 18
* Angular Material
* Tailwind CSS
* RxJS
* Chart.js

---
